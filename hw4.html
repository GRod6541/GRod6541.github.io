Lists Usage
Unordered List Are used to list items using bullets. The tags < ul > are used to identify unordered lists
Ordered List Are used to list items using numbers as specified by user, the default one is numerical numbers like 1. An ordered list starts with the <ol>; tag. Each list item starts with the <li> tag.
Definition List The <dl> tag defines a description list. The <dl> tag is used together with <dt> (defines terms/names) and <dd> (describes each term/name).
HTML
Hyper Text Markup Language. It is the standard markup language for Web pages.
CSS
Cascading Style Sheet. It is is the language we use to style an HTML document.
FTP
File Transfer Protocol. is a standard communication protocol used for the transfer of computer files from a server to a client on a computer network
SSH
Secure Shell. It is is a cryptographic network protocol for operating network services securely over an unsecured network
