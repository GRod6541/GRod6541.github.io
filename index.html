What is Python

Python is a programming language created by Guido Van Rossum
and officially released in 1991.

It has numerous applications such as in web development, in mathemetics, in data scienc, in system scripting, and in software developement among many others.

Python Syntax

Python was primarily designed for easy readability. It has numerous similarities with natural languages like English. It has also received influences from mathematics.

Unlike other programming languages which use semicolon or parenthesis to complete commands, Python uses new lines instead.
Python has identation, to define scope like the ones found in loops, functions and classes, while other programming languages use curly braces.
